# Documentación: Fase 2 - Vida Práctica PRO

Esta fase transforma la aplicación en un sistema integral de gestión de vida, optimizado para la eficiencia personal, el ahorro y la calidad del tiempo familiar.

## 1. Módulo: Custodia e Hija
Gestiona el tiempo compartido y las responsabilidades parentales.
- **Calendario:** Diferencia días conmigo, eventos escolares, citas médicas y vacaciones.
- **Gastos Hija:** Seguimiento específico de gastos relacionados con la niña para control financiero claro.

## 2. Módulo: Planes Alicante PRO
Base de datos local y editable de actividades.
- **Filtros Inteligentes:** Permite buscar planes aptos para lluvia, con niños o por categorías de precio (Gratis/Barato/Medio).
- **IA Ready:** Los planes alimentan al asistente IA para sugerencias contextuales.

## 3. Módulo: Alimentación PRO (Menú y Compra)
Sistema interconectado para optimizar el tiempo en la cocina y el dinero en el súper.
- **Menú Semanal:** Planifica Desayuno, Comida y Cena. Marca días específicos con tu hija.
- **Generación Automática:** Con un clic, los platos planeados se convierten en una lista de ingredientes en la Lista de Compra.
- **Compra PRO:** Categorización automática y control de precios estimados.

## 4. Módulo: Presupuesto PRO
Control financiero avanzado.
- **Semáforo Económico:** Indicador visual (Verde/Amarillo/Rojo) basado en el dinero disponible semanal tras descontar gastos fijos, deudas y meta de ahorro.
- **Dinero Libre:** Te dice exactamente cuánto puedes gastar "hoy" sin comprometer el mes.

## 5. Módulo: Gestión de Estrés (Modo Emergencia)
Interfaz de "Foco Absoluto".
- **Reducción de Ruido:** Oculta todo excepto las 3 tareas más críticas y la información de emergencia.
- **Contactos y Docs:** Acceso rápido a teléfonos y documentos vitales.

## 6. Revisión Semanal
Proceso dominical para cerrar la semana.
- **Estadísticas:** Resumen de tareas completadas, tiempo con hija y gasto total.
- **Planificación:** Recomendaciones automáticas para la semana entrante.

## 7. Mejoras en el Asistente IA
Nuevos botones optimizados para la Fase 2:
- **"Estoy agobiado":** Pide a la IA que elija por ti las 3 prioridades del día.
- **"Plan barato hija":** Busca específicamente en tu base de Alicante opciones sin coste.
- **"Menú barato":** Optimiza tus comidas para ahorrar al máximo.
